   <P><Strong>"openapi": "3.0.3": </Strong> berfungsi untuk mendefinisikan versi OpenAPI yang digunakan dalam spesifikasi ini.</P>
   <P><Strong>"info":</Strong> berfungsi untuk memberikan informasi metadata tentang API. Bagian yang termasuk dalam metadata API adalah judul, versi, deskripsi, alamat situs web untuk syarat dan ketentuan penggunaan, informasi kontak, dan lisensi. </P>

"title": Judul API.
"version": Versi API.
"description": Deskripsi singkat tentang API.
"termsOfService": URL yang mengarah ke syarat dan ketentuan penggunaan API.
"contact": Informasi kontak tim pengembang API, termasuk nama, alamat email, dan URL GitHub.
"license": Informasi tentang lisensi API, termasuk nama dan URL lisensi.
"servers": Ini adalah bagian yang menentukan server atau lingkungan dimana API dapat diakses.

"description": Deskripsi server.
"url": URL dasar API. {environment} adalah variabel yang harus diganti dengan nilai dari lingkungan (misalnya: development, production, staging).
"variables": Variabel yang dapat digunakan dalam URL. Dalam kasus ini, hanya ada satu variabel yaitu "environment" yang dapat memiliki nilai dari "development", "production", atau "staging".
